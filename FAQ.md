## English

### General Questions

#### What is this archive?

This is a preservation project dedicated to archiving Java mobile games from Gameloft, particularly focusing on different versions, screen resolutions, and modifications.

#### Why archive these games?

Many classic mobile games are being lost to time as old devices become obsolete. This archive helps preserve gaming history for future generations to study and appreciate.

#### Are these games free?

The games are provided for archival and educational purposes only. All rights belong to Gameloft and respective copyright holders.

#### Can I play these games?

Yes, if you have a compatible Java ME device or emulator. Popular emulators include:
- J2ME Loader (Android)
- KEmulator
- MicroEmulator

### Technical Questions

#### What file format are the games in?

All games are in `.jar` (Java Archive) format for Java ME (J2ME) platform.

#### What devices are compatible?

Any device that supports Java ME (J2ME) applications, including:
- Feature phones from Nokia, Samsung, Sony Ericsson, etc.
- Modern Android devices using J2ME emulators

#### How do I install these games?

1. Download the appropriate JAR file for your device's resolution
2. Transfer to your phone via Bluetooth, USB, or memory card
3. Open and install the JAR file
4. For Android: Use J2ME Loader app

#### What do the version numbers mean?

Version numbers follow the format `v[major].[minor].[patch]`:
- `v1.0.0` - Initial release
- `v1.1.0` - Minor update with new features
- `v1.1.1` - Bug fixes and small improvements

#### Which resolution should I choose?

Check your device's screen resolution in the settings. Common resolutions:
- 128x160 - Very small screens
- 176x208/176x220 - Small screens
- 240x320 - Medium screens (most common)
- 320x240 - Landscape medium screens

#### What's the difference between touchscreen and keypad versions?

- **Touchscreen**: Designed for devices with touch input
- **Keypad**: Designed for traditional button-based phones

#### What does "modified" mean?

Modified (modded) versions may include:
- Unlocked levels
- Unlimited resources
- Bug fixes
- Enhanced features
- Language changes

### Contributing

#### How can I contribute?

See [CONTRIBUTING.md](./CONTRIBUTING.md) for detailed guidelines.

#### I have a game version not in the archive. What should I do?

1. Check if it's already listed
2. Open an issue with game details
3. If you can provide the file, follow the contribution guidelines

#### Can I add games from other developers?

Currently, this archive focuses exclusively on Gameloft games.

## Русский

### Общие вопросы

#### Что такое этот архив?

Это проект по сохранению, посвященный архивированию Java мобильных игр от Gameloft, с особым акцентом на различные версии, разрешения экрана и модификации.

#### Зачем архивировать эти игры?

Многие классические мобильные игры теряются со временем, так как старые устройства устаревают. Этот архив помогает сохранить историю игр для будущих поколений.

#### Эти игры бесплатные?

Игры предоставляются только в архивных и образовательных целях. Все права принадлежат Gameloft и соответствующим правообладателям.

#### Могу ли я играть в эти игры?

Да, если у вас есть совместимое устройство Java ME или эмулятор. Популярные эмуляторы:
- J2ME Loader (Android)
- KEmulator
- MicroEmulator

### Технические вопросы

#### В каком формате игры?

Все игры в формате `.jar` (Java Archive) для платформы Java ME (J2ME).

#### Какие устройства совместимы?

Любое устройство, поддерживающее приложения Java ME (J2ME), включая:
- Кнопочные телефоны Nokia, Samsung, Sony Ericsson и др.
- Современные Android устройства с эмуляторами J2ME

#### Как установить эти игры?

1. Скачайте подходящий JAR файл для разрешения вашего устройства
2. Перенесите на телефон через Bluetooth, USB или карту памяти
3. Откройте и установите JAR файл
4. Для Android: используйте приложение J2ME Loader

#### Что означают номера версий?

Номера версий следуют формату `v[основная].[дополнительная].[патч]`:
- `v1.0.0` - Первый релиз
- `v1.1.0` - Небольшое обновление с новыми функциями
- `v1.1.1` - Исправление ошибок и небольшие улучшения

#### Какое разрешение выбрать?

Проверьте разрешение экрана вашего устройства в настройках. Распространенные разрешения:
- 128x160 - Очень маленькие экраны
- 176x208/176x220 - Маленькие экраны
- 240x320 - Средние экраны (наиболее распространенные)
- 320x240 - Средние экраны в альбомной ориентации

#### В чем разница между сенсорными и клавишными версиями?

- **Сенсорный экран**: Разработаны для устройств с сенсорным вводом
- **Клавиатура**: Разработаны для традиционных кнопочных телефонов

#### Что означает "модифицированная"?

Модифицированные (моддированные) версии могут включать:
- Открытые уровни
- Неограниченные ресурсы
- Исправление ошибок
- Улучшенные функции
- Изменения языка

### Вклад в проект

#### Как я могу внести вклад?

Смотрите [CONTRIBUTING.md](./CONTRIBUTING.md) для подробных инструкций.

#### У меня есть версия игры, которой нет в архиве. Что делать?

1. Проверьте, не указана ли она уже
2. Откройте issue с деталями игры
3. Если можете предоставить файл, следуйте руководству по внесению вклада

#### Могу ли я добавить игры от других разработчиков?

В настоящее время этот архив сосредоточен исключительно на играх Gameloft.
