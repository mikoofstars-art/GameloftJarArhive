# Wonder Zoo v1.0.0

## English

### Version Details

- **Version**: 1.0.0
- **Release Date**: TBA
- **Language**: Russian
- **Developer**: Gameloft

### Available Builds

#### 240x320
- **Touchscreen**: Place JAR file in `240x320/touchscreen/` directory
- **Keypad**: Place JAR file in `240x320/keypad/` directory

### Game Features

- Build and manage your own zoo
- Rescue exotic animals
- Care for animals and keep them happy
- Expand your zoo with new attractions

### Installation

1. Download the appropriate JAR file for your device
2. Transfer to your phone via Bluetooth, USB, or memory card
3. Install and run the JAR file
