### About

This repository is dedicated to preserving different versions of Gameloft Java games (.jar) for feature phones. The archive aims to collect all available versions, screen resolutions, and modifications of classic mobile games.

### ‚ö†Ô∏è Important Notices

- **Language**: Most games in this archive are in Russian language
- **Modified versions**: This archive may contain modified (modded) versions of games
- **Educational purposes**: This archive is for preservation and educational purposes only
- **Copyright**: All rights belong to Gameloft and respective owners

### üéØ Current Focus: Wonder Zoo

The archive currently focuses on collecting all versions of **Wonder Zoo**, including:

- **Versions**: From 1.0.0 to the latest available
- **Screen resolutions**: All supported resolutions (128x128, 128x160, 176x208, 240x320, etc.)
- **Control types**:
  - Touchscreen versions
  - Non-touchscreen (keypad) versions
- **Modifications**:
  - Original (unmodified) versions
  - Modified (modded) versions
